# KeyVision ⌨️✨

KeyVision은 컴퓨터 수업 시 강사의 키보드 입력을 실시간으로 시각화하여 학생들의 단축키 학습을 돕는 **웹 기반 교육용 도구**입니다.

## 🌟 프로젝트 소개 (Overview)
학교나 학원 등 교육 현장에서 단축키를 가르칠 때, 말이나 자막만으로는 학생들이 직관적으로 이해하기 어려운 경우가 많습니다. KeyVision은 강사가 누르는 키를 대형 화면에 즉각적으로 하이라이트하여 보여줌으로써 학생들의 이해도와 몰입감을 높여줍니다. 특히 **Windows, Mac, Chromebook** 등 교육 현장에서 쓰이는 다양한 운영체제(OS)의 키보드 레이아웃을 모두 지원합니다.

## ✨ 주요 기능 (Features)
- **실시간 입력 시각화**: JavaScript `keydown`/`keyup` 이벤트를 활용해 누른 키를 즉각적으로 화면에 강조(Highlight)합니다.
- **다중 OS 레이아웃 지원**: 클릭 한 번으로 Windows, Mac, Chromebook 키보드 레이아웃을 자유롭게 전환할 수 있습니다.
- **프리미엄 디자인**: 다크 모드(Dark Mode)와 글래스모피즘(Glassmorphism) 기반의 유려한 UI 디자인을 적용하여 학생들의 시선을 사로잡습니다.
- **철저한 디테일 고증**: Mac 키보드의 `return`, `delete`, `caps lock` 표기나 Chromebook의 상단 특수 액션 키(검색, 런처 등)까지 실제 키보드와 100% 동일하게 물리적/시각적으로 구현했습니다.

## 🎓 강사 활용 가이드 (Instructor Guide)

KeyVision은 순수 웹 어플리케이션(Web App)이므로, 별도의 무거운 프로그램 설치 없이 링크 접속만으로 즉시 사용할 수 있습니다.
하지만 웹 브라우저의 보안 정책상, 강사의 물리적 키보드에서 `Windows` 키나 `Alt+Tab` 같은 시스템 단축키를 누르면 브라우저를 벗어나 컴퓨터 자체의 시스템 메뉴가 열려버리는 한계가 있습니다.

이를 방지하기 위해 KeyVision에는 **"안전한 우회 키 (Safe Aliases)"** 매핑 기술이 탑재되어 있습니다. 강사님은 아래의 우회 규칙만 기억하시면 운영체제의 방해 없이 우아하게 시연하실 수 있습니다.

### 🛡️ 시연용 우회 키(Alias) 사용 팁
**1. Windows 레이아웃 시연 시**
- `Windows 로고 키` 보여주기: 진짜 Win 키 대신 강사님 키보드의 **`우측 Ctrl (Right Control)`**을 누르시면 화면 상의 Windows 키에 안전하게 불이 들어옵니다.

**2. Mac 레이아웃 시연 시**
- `Command (⌘)` 보여주기: 강사님 키보드의 **`우측 Ctrl (Right Control)`**을 누르시면 됩니다.
- `Option (⌥)` 보여주기: 물리적 **`Alt`** 또는 **`Menu(우클릭)`** 키를 누르시면 됩니다.

**3. Chromebook 레이아웃 시연 시**
- `Search (🔍)` 보여주기: 크롬북의 돋보기 키는 일반 키보드의 `CapsLock` 위치에 있습니다. 강사님 키보드의 **`CapsLock`**을 켜거나 **`우측 Ctrl`**을 누르시면 돋보기 키에 직관적으로 불이 들어옵니다.
- `상단 특수 키 (⬅️, 🔆, 🔇 등)` 보여주기: 강사님 키보드의 **`F1` ~ `F10`** 키를 순서대로 누르시면 화면의 특수 아이콘 열과 완벽히 1:1로 매칭되어 반응합니다.

## 🚀 시작하기 및 배포 (Deployment)

본 프로젝트는 정적 자산(HTML, CSS, JS, SVG)으로만 구성되어 있어 **GitHub Pages**를 통해 매우 쉽게 배포 및 활용이 가능합니다. 모든 키보드의 SVG 정보는 자바스크립트에 안전하게 번들링되어 있으므로 CORS 이슈가 없습니다.

1. `index.html` 파일을 다운로드하여 로컬에서 더블 클릭하면 인터넷이나 로컬 서버 없이도 브라우저에서 바로 동작을 확인할 수 있습니다.
2. GitHub Repository에 본 코드를 업로드(Push)한 뒤, `Settings > Pages` 에서 **GitHub Actions**를 Source로 지정해 두면 설정해둔 `deploy.yml`을 통해 자동으로 전 세계 어디서든 접속 가능한 웹사이트로 배포됩니다.
